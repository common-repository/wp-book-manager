== Description ==

WP Book Manager plugin allows you to manage books and their informations in a website. The books can have the following meta informations to manage:

Book Name
Book Description
Author Name
Publisher Name
Genre
Publisher Website
Published Date
Book Price
Book Sale price
Book ISBN


Templates include:

single-book.php
archive-book.php

== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **WP Book Manager** in the search box and click on search button.
3. Find WP Book Manager plugin.
4. Then click on Install Now after that activate the plugin.
